== Description ==

Qurify and display whatever QR Code you want and (optional) let your visitors qurify their own QR Codes, directly on your blog!
**Features**

    *  Simple
    *  Customizable
    *  Widget support
    *  No options page (yes, its a feature)
    *  Uses Wordpress resources (no extra files needed)

**Usage**

If you use WordPress widgets, just drag the widget into your sidebar and configure.
Be sure to have activated the plugin in your Plugin Settings.

== Installation ==

Drop the qurify-wordpress folder (or only the .php file) into /wp-content/plugins/ and activate the plug in the Wordpress admin area. Add the widget at Appearance / Widgets and save your settings.
